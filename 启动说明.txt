==========================================
     小说 AI 写作助手 - 启动说明
==========================================

【方法一：双击启动（推荐）】

1. 第一次使用前，请确保已安装 Node.js:
   - 访问 https://nodejs.org/ 下载 LTS 版本
   - 按照提示完成安装

2. 启动方式（三选一）：

   方式 A - 使用批处理文件（最简单）:
   - 双击运行 "启动小说写作助手.bat"
   
   方式 B - 使用 PowerShell（推荐，界面更好看）:
   - 右键点击 "启动小说写作助手.ps1"
   - 选择 "使用 PowerShell 运行"
   
   方式 C - 创建桌面快捷方式:
   - 右键 "启动小说写作助手.bat"
   - 选择 "发送到" -> "桌面快捷方式"
   - 以后直接双击桌面图标即可启动

3. 等待启动完成，浏览器会自动打开 http://localhost:3000
   如果没有自动打开，请手动在浏览器中访问该地址


【方法二：命令行启动（开发者使用）】

如果您熟悉命令行，也可以：

1. 打开命令提示符或 PowerShell
2. 进入项目目录: cd my-app
3. 安装依赖: npm install（首次需要）
4. 启动服务: npm run dev


【常见问题】

Q: 提示 "未检测到 Node.js"
A: 请先访问 https://nodejs.org/ 下载并安装 Node.js

Q: 启动后浏览器显示 "无法访问此网站"
A: 请等待命令行窗口显示 "Ready" 或 "Local: http://localhost:3000" 后再访问

Q: 如何停止服务器？
A: 在命令行窗口中按 Ctrl+C，然后按 Y 确认

Q: 首次启动很慢？
A: 首次启动需要下载依赖，请耐心等待，后续启动会很快


【系统要求】

- Windows 10 或更高版本
- Node.js 18.0 或更高版本
- 至少 4GB 内存
- 需要联网（用于 AI 生成功能）
